# Code Block Syntax Highlighting Upgrade

## Problem
Current regex-based `SyntaxHighlighter.swift` produces poor highlighting. Limited token types, no line numbers, no copy button.

## Solution
Replace with **HighlighterSwift** (actively maintained, built-in line numbering, theme support).

## Changes

### 1. Add dependency â€” `Package.swift`
```swift
.package(url: "https://github.com/smittytone/HighlighterSwift.git", from: "3.1.0")
```
Add `"HighlighterSwift"` to target dependencies.

### 2. New file â€” `Sources/CourseReader/Services/HighlighterService.swift`
Actor-isolated wrapper:
- `@MainActor final class HighlighterService`
- Singleton `shared`
- `init()`: create `Highlighter()`, set theme to `"dark-plus"` or `"atom-one-light"` based on appearance
- `highlight(_ code: String, language: String?, fontSize: CGFloat) -> NSAttributedString`
  - Configure `theme.setCodeFont(NSFont.monospacedSystemFont(ofSize: fontSize, weight: .regular))`
  - Configure `theme.lineSpacing` and `theme.paraSpacing`
  - Call `highlighter.highlight(code, as: language)` â€” returns `NSAttributedString` with colors applied
- `updateTheme(for view: NSView)`: detect dark/light, call `setTheme()`

### 3. Modify â€” `Sources/CourseReader/Views/LessonView.swift`

#### a. Replace `SyntaxHighlighter.highlight()` calls (lines 276, 418)
```swift
// BEFORE
let highlighted = SyntaxHighlighter.highlight(code, language: lang)

// AFTER
let highlighted = await HighlighterService.shared.highlight(code, language: lang, fontSize: codeFontSize)
```

#### b. Make `parseMarkdown` async
```swift
private func parseMarkdown(_ md: String, fontSize: Double) async -> (NSAttributedString, [NSRange])
```

#### c. Update `updateNSView` to use Task
```swift
Task { @MainActor in
  let (attrs, codeRanges) = await parseMarkdown(markdown, fontSize: fontSize)
  textView.textStorage?.setAttributedString(attrs)
  textView.codeBlockRanges = codeRanges
  textView.setNeedsDisplay(textView.bounds)
}
```

#### d. Add line numbers via `LineNumberData`
HighlighterSwift has built-in line numbering. No custom gutter needed.

#### e. Add copy button to toolbar
```swift
ToolbarItem {
  Button(action: { copyCodeToClipboard() }) {
    Label("Copy Code", systemImage: "doc.on.doc")
  }
}
```

### 4. Delete â€” `Sources/CourseReader/Helpers/SyntaxHighlighter.swift`
No longer needed.
