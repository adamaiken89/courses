# Plan: New CourseReader App Icon — Book Stack

## Goal
Replace current open-book icon with a **stacked books** design in `scripts/make-icon.swift`.

## Design Concept
- **Background**: Rounded-rect with blue-to-purple gradient (ties into SVG logo palette)
- **Books**: 3 books stacked at slight angles, fanned out like a study pile
  - Bottom book: deep blue, horizontal
  - Middle book: medium blue-purple, tilted ~8° left
  - Top book: light purple, tilted ~5° right
- **Details**: Each book has a spine edge (darker shade), pages visible as thin white line on the open edge
- **Shadow**: Drop shadow under the stack for depth (same technique as Decompress/current icon)
- No bookmark, no text lines — clean book shapes only

## Files to Modify

### `scripts/make-icon.swift` (rewrite drawing code)
- Keep: CGContext setup (1024×1024), gradient background, PNG export
- Replace: book drawing section (lines 34–93) with stacked-books geometry
- Keep: red bookmark removal (new design has no bookmark)
- Output: writes `icon_1024.png` to `$ARGV[1]`

### No other files change
- `scripts/make-app-bundle.sh` already calls `make-icon.swift` → sips resize → iconutil — no changes needed

### Book geometry (all relative to 1024×1024 canvas)
- Canvas center: (512, 512)
- Books sized ~320×240, stacked with ~20px vertical offset each
- Rotation via CGContext.rotate (save/restore GState around each book)
- Bottom book: no rotation, y = 340
- Middle book: rotate -8°, y = 370  
- Top book: rotate +5°, y = 400
